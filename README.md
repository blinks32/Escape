# Escape – Plate Tectonics Educational Game

**A frogger-style adventure where players dodge earthquakes and hazards while learning about plate tectonics.**

Built for **Legends of Learning** platform as an interactive K-12 science game.

### Learning Objective
- Understand **plate tectonics**: How tectonic plates move, cause earthquakes, and shape Earth's surface.
- Players avoid falling debris, cracks, and seismic events in a side-scrolling/jumping challenge.

### Gameplay
- Frogger-inspired mechanics: Jump across moving platforms, dodge hazards, and navigate dangerous terrain.
- Real-time decision-making tied to plate boundary concepts (convergent, divergent, transform).
- Engaging for middle school science standards.
